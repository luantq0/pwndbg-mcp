# Security Policy

## Reporting a Vulnerability

Please report security issues **privately** — do not open a public issue.

- Preferred: GitHub's private reporting at
  **Security → Report a vulnerability** on this repository.

Please include a description, affected version/commit, and a minimal
reproduction if possible. You can expect an initial response within a few days.
Confirmed issues will be fixed in the latest release and credited unless
you prefer to remain anonymous.
